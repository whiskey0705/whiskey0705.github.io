webstrom快捷键

  • 选中多行，进行输入或修改 alt + 鼠标左键按住下拉
  • 打开目录下任意文件 ctrl + shift + n
  • 通过字符快速定位（可以查class、id） ctrl + shift + alt + n
  • 开打/关闭侧边栏 alt + 1（数字1）
  • 在打开的多个标签之间切换 alt + 方向键left/right
  • 删除行 ctrl + x
  • 复制行 ctrl + d
  • 查找行 ctrl + g
  • 代码上移/下移 ctrl + shift + 方向键up/down
  • 快速定位错误或警告的高亮代码 F2
  • 高亮显示所选文本（按ESC取消高亮，一次只能选一行） 先选中文本，再ctrl + shift + F7
  • 快速打开光标处的类、方法（函数） ctrl + b
  • 在光标当前所在行的下方新建行 shift + enter
  • 在光标当前所在行的上方新建行 ctrl + alt + enter
  • 用包含选中的代码行（包括if、while…） ctrl + alt + t
  • 格式化代码（类似codepen里的tidy） ctrl + alt + l
  • 自动生成注释,在函数上一栏输入/**后按回车
  • search everywhere 双击shift
  • 打开最近文件 ctrl + e
