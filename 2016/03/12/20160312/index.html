sublime快捷键

发表于 2016-03-12   |   分类于 前端   |  

go to anything ctrl + p

  • @ 符号跳转：输入@symbol跳转到symbol符号所在的位置
  • # 关键字跳转：输入#keyword跳转到keyword所在的位置
  • : 行号跳转：输入:12跳转到文件的第12行。

多行光标

  • ctrl + d选择当前光标所在的词，ctrl + k会跳过下一个,ctrl + u撤销上一次操作
  • 按住ctrl+鼠标左键也能创建多行光标

命令模式 ctrl + shift + p

其他操作

  • 在光标所在行的下一行添加内容 // ctrl + enter
  • 在光标所在行的上一行添加内容 // ctrl + shif t+ enter
  • 保持代码缩进格式的粘帖 // ctrl + shift + v
  • 逐词移动 // ctrl + ←/→，逐词选择 // ctrl + shift + ←/→
  • Ctrl + ↑/↓移动当前显示区域，Ctrl + Shift + ↑/↓移动当前行
  • jsFormat 快捷键ctrl + alt + f
