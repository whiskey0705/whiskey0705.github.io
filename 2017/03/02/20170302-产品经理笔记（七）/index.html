线框图

为了构建一个好的产品，你需要从概念上结构化它，线框图就是一个很好的工具。

  • 线框图是一种视觉导航，可以勾勒出产品的结构图
  • 收到反馈信息后，添加更多的细节
  • 在小团队里，产品经理需要自己做线框图
  • 线框图让沟通变得更简单

线框图(wireframes)、模型(mockups)、原型（prototypes）之间的区别

  • Wireframes

    • 最新绘制
    • 只绘制最基本的框架和布局
    • 精确度低

  • Mockups

    • 静态呈现产品的最终形态
    • 有更多的细节和色彩
    • 由设计师完成

  • Prototypes

    • 考虑可使用性
    • 有基础的交互和高精度
    • 在使用过程中可以看出潜在的问题
