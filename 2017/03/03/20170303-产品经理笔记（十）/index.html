发表于 2017-03-03   |   分类于 产品经理   |  

沟通

作为一个产品经理，沟通是一项必不可少的技能，学习沟通的艺术。

  • 会议

    • 如何利用会议
    • 确保跟进会议上提出的议题
    • 给每个人发一份回顾的会议纪要

  • 自信

    • 注意语调以及确保信息传达准确

  • 邮件

    • 简明、表达清晰

  • 非正式会议

与工程师沟通

在与工程师的沟通中，应该尽量细节化，包括技术方面的信息。不要向对待代理一样对待工程师，注意聆听他们的反馈意见。

与设计师沟通

  • 给予足够的创作自由
  • 多沟通
  • 全程向设计师提供用户数据
  • 不要指挥设计师如何做
  • 永远先描述用户遇到的问题，让设计师想解决方法

与老板沟通

  • 简洁是首要秘诀
  • 用商业效果说话
  • 用对方的方式交流
