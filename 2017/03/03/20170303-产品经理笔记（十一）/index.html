发表于 2017-03-03   |   分类于 产品经理   |  

产品经理要学技术的3个理由

  1. 知道实际要构建的产品是怎么样的
  2. 可以跟好的和工程师打交道
  3. 可以帮助你更好的做决定

internet如何运转

  • Server： 数据储存在数据库，根据请求发送相应的数据到客户端
  • Client： 用来访问服务器的客户端口
  • 每个服务器都有一个IP地址
  • DNS（Domain Name System）： 负责域名和IP地址之间的相互解析

前端、后端、技术栈

  • 前端：用户看见的交互的部分

    • 颜色、文字
    • HTML\CSS\JS

  • 后端： 看不见的部分

  • 应用层： 通过编码将数据分配到指定的位置

  • 技术栈： 做一个APP需要的一整套技术，包括前后端

什么是API

Application Programming Interface, API称为应用编程接口，就是软件系统不同组成部分衔接的约定
